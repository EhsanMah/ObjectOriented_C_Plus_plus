Detailed Description

Ranger will be used as an abstract class for different type of rangefinder sensor classes .

this class consists of basic features of a rangefinder sensor which includes:
Orientation Offset, Field of View, Angular Resolution, Number of Samples, Sensing Type, Data
classes inherited from this class can conduct computations by adding functions for the features that are exclusiveto them

Constructor & Destructor Documentation

Ranger::Ranger ( )

The Default constructor sets all the attributes to zero. Inherited Child classes are suppose to configure these Parameters
Besides it initiates a seed and a random number generator which can be used by child classes to generate data .

Note
NOTE: All the member variables should be set to default values in child classes
